# TRUSTed Web Architecture

Public marketing website for TRUSTed Web Architecture: custom website services and a guided, expert-supported website creation system.

## Local development

```bash
npm install
npm run dev
```

Build the production site with:

```bash
npm run build
```

## Deployment

This project is configured for Vercel's Next.js framework detection. Import the GitHub repository in Vercel and deploy with the default settings. No environment variables, custom domains, or DNS changes are required.
