# Codex Intake Export

The local lead mock creates a Markdown brief in memory. It excludes name, email, phone, and other unnecessary personal contact data. It records the assessment, recommendations, roadmap, preliminary estimate, assumptions, unknowns, and requested timeline.

Every brief defaults to `NEW — FOUNDER REVIEW REQUIRED` and includes explicit restrictions against implementation, client contact, repository creation, private-system access, proposal issuance, and deployment. Only a separate founder-controlled `APPROVED_FOR_BUILD` decision can authorize work; the public website cannot change that status.

Do not write public submissions into this repository or grant the website access to the Codex workspace. A future secure lead system may provide an opaque internal record reference.
