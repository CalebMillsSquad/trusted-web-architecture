# Privacy and Retention Notes

- The browser stores only non-contact assessment state and recent display messages in session storage for navigation continuity.
- Contact fields remain form-local and are not persisted client-side.
- Free-form AI messages and structured assessment context are sent to OpenAI with application response storage disabled.
- No database, CRM, analytics event, file write, or authoritative lead record is created.
- No public website credentials, API keys, payment data, health data, legal secrets, or confidential business information should be entered.
- Local mock lead data exists only in request/response memory and is not a production retention design.

Production activation requires qualified privacy/legal review, an approved data inventory, purposes, consent language, provider terms, access rules, retention/deletion schedule, incident and recovery procedures, and an updated public notice.
